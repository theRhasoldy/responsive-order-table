# Responsive Order Table

A modern, responsive order management dashboard built with React Router 7, Tailwind CSS 4, and Shadcn UI. This project demonstrates a robust filtering system, dark mode support, and a high-performance data table.

## 🚀 Features

- **Dynamic Filtering System**:
  - Centralized filter configuration for easy extensibility.
  - Search by multiple criteria (ID, Customer Name).
  - Filter by status with an "All" reset option.
  - Interactive filter chips (badges) to quickly see and remove active filters.
- **Smart Data Table**:
  - Responsive layout that adapts to different screen sizes.
  - Type-safe row identifiers and column definitions.
  - Empty state handling.
- **Modern Tech Stack**:
  - **React Router 7**: Utilizing the latest features.
  - **Tailwind CSS 4**: Modern, high-performance styling.
  - **next-themes**: Robust dark mode support with server-side hydration matching.
  - **Phosphor Icons**: A consistent and beautiful icon set.
  - **Radix UI**: Accessible primitive components.

## 🛠️ Tech Stack

- **Framework**: [React Router 7](https://reactrouter.com/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **UI Components**: [Shadcn UI](https://ui.shadcn.com/) (Radix UI based)
- **Icons**: [Phosphor Icons](https://phosphoricons.com/)
- **Type Safety**: [TypeScript](https://www.typescriptlang.org/)

## 📦 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v20 or higher recommended)
- [npm](https://www.npmjs.com/)

### Installation

1. Clone the repository:

   ```bash
   git clone <your-repo-url>
   cd responsive-order-table
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

### Development

Start the development server:

```bash
npm run dev
```

The app will be available at `http://localhost:5173`.

### Type Checking

To run type checking and generate React Router types:

```bash
npm run typecheck
```
